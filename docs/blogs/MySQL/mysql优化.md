### MySQL储存引擎

| 对比项   | MyISAM                                                   | InnoDB                                                       |
| -------- | -------------------------------------------------------- | ------------------------------------------------------------ |
| 主外键   | 不支持                                                   | 支持                                                         |
| 事务     | 不支持                                                   | 支持                                                         |
| 行表锁   | 表锁，即使操作一条记录也会锁住整个表，不适合高并发的操作 | 行锁，操作时只锁一行，不对其他行有影响，适合高并发的操作     |
| 缓存     | 只缓存索引，不缓存真实数据                               | 不仅缓存索引还缓存真实数据，对内存要求较高，而且内存大小对性能有绝对性影响 |
| 表空间   | 小                                                       | 大                                                           |
| 关注点   | 性能                                                     | 事务                                                         |
| 默认安装 | Y                                                        | Y                                                            |
